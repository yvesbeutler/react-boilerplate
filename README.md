# react-boilerplate

**Author:** Yves Beutler
**Github:** [github.com/yvesbeutler](https://github.com/yvesbeutler)

**Version:** 0.1
 
Simple boilerplate to get started with your react application.

Usage
---
Start the development server with this command:
```
npm run start
```

Setup
---
 
```
npm install
```

Compile
---
```
npm run build
```

Project Structure
---

```
.
├── app                      # All source code
├── dist                     # Output directory
```
